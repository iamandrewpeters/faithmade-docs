# Event Location Options

Not every event happens at your building. Some are at the park, some are at someone's house, and some are on Zoom. Faithmade gives you four location modes so your people always know where to show up.

## The Four Options

Find the **Location** dropdown when creating or editing an event:

1. **At Church** — pulls your default address from Settings automatically. Sunday services, midweek gatherings, easy.
2. **Saved Location** — pick from spots you use regularly (the park pavilion, the community center). Save it once, reuse forever.
3. **Custom Location** — for one-off spots. Type in the address for that retreat center or the pastor's barn.
4. **Online / Livestream** — for virtual events. No address needed. Just make sure your join link is somewhere obvious in the description.

## Tips That Actually Help

- If **At Church** shows the wrong address, update it in **Settings > Church Info**.
- For tricky locations, add directions in your event description. Your first-timers will thank you.
- For online events, put the Zoom or livestream link in the description or registration field. Don't make people hunt for it.
