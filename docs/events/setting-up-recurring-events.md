# Setting Up Recurring Events

Wednesday night Bible study doesn't need to be re-created every single week. Set it up once, pick your recurrence, and let Faithmade handle the rest.

## How to Set It Up

When creating or editing an event, scroll to the **Schedule** section. You'll see a **Recurrence** dropdown with three options:

1. **Does Not Repeat** (default) — for one-time events
2. **Weekly** — pick the day. Tuesday night prayer meeting? Weekly, Tuesday, done.
3. **Monthly** — pick the week + day. Men's breakfast the first Saturday? Monthly, First, Saturday. Done.

## Setting an End Date

You've got three choices:
- **Never** — for ongoing services
- **On a specific date** — for a 10-week Bible study
- **After X occurrences** — for a 6-session class

:::tip
Not sure when it ends? Pick "Never" and update it later.
:::

## Editing a Recurring Event

Changes apply to the whole series going forward. Need to skip one week? Edit or delete that single occurrence without messing up the rest.

Recurring events share the same title, description, and image across all occurrences. Update once, it updates everywhere.
